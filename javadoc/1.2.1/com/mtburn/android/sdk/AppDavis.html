      • createInstreamAdAdapter

        public static <T extends ListAdapter> ADVSInstreamAdAdapter<T> createInstreamAdAdapter(Context context,
                                                                       T originalAdapter,
                                                                       java.lang.String adSpotId,
                                                                       int adCount,
                                                                       java.util.List<java.lang.Integer> positions)
        `ADVSInstreamAdAdapter` is for loading instream type advertisements. It's recommended for app using ListView / ListAdapter to invoke this method.
        パラメータ:
        context - Application context object.
        originalAdapter - Adapter to be binded
        adSpotId - adSpotId to be registered in advance
        adCount - advertisement information count to be received in the callback. If 0 is given, pre-registered is selected.
        positions - each ad position to be inserted into table in order to control ad position. If null is given, pre-registered is selected.
        関連項目:
        ADVSInstreamAdAdapter
